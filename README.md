# AirBNB-analysis# Airbnb Exploratory Tableau Analysis

## Overview
This project explores Airbnb listing data using Tableau to understand pricing patterns, availability trends, and geographic distribution of listings.

## Objective
- Explore Airbnb pricing and availability across locations
- Practice data visualization and dashboard development in Tableau

## Tools
- Tableau
- Microsoft Excel

## Notes
This project was created as a learning exercise to practice Tableau visualization techniques. It is exploratory in nature and does not include advanced KPI analysis or business recommendations.

## Files
- Tableau workbook (.twb)
- Source dataset (Excel)
- Dashboard screenshots
